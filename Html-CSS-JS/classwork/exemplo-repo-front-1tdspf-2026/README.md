# ESPORTE 🚀

## HISTÓRIA DOS ESPORTES
O esporte é uma parte fundamental da cultura humana, remontando a milhares de anos. Desde as antigas civilizações, como os gregos e romanos, até os dias atuais, o esporte tem desempenhado um papel crucial na sociedade. Ele promove a saúde física, a camaradagem e a competição saudável entre as pessoas. O esporte também tem o poder de unir as pessoas, transcender barreiras culturais e inspirar gerações. Com o avanço da tecnologia, o esporte evoluiu para incluir uma variedade de modalidades, desde os esportes tradicionais, como futebol e basquete, até os esportes eletrônicos, que estão ganhando popularidade em todo o mundo. O esporte continua a ser uma força poderosa na vida das pessoas, promovendo a saúde, a diversão e a conexão social.

## Exercícios

1. Crie uma nova página HTML chamada `voley.html` dentro da pasta `paginas` e adicione um conteúdo relacionado ao vôlei. Por exemplo, como uma breve história do esporte, regras básicas ou informações sobre jogadores famosos.

2. Adicione um link para a nova página de vôlei no menu de navegação do arquivo `index.html`, para que os visitantes possam acessar facilmente a página de vôlei a partir da página inicial.

3. Crie um arquivo CSS chamado `voley.css` dentro da pasta `css` e adicione estilos personalizados para a página de vôlei. Por exemplo, você pode escolher uma paleta de cores relacionada ao esporte ou adicionar imagens de jogadores de vôlei como plano de fundo.
4. Certifique-se de vincular o arquivo CSS `voley.css` à página `voley.html` para que os estilos sejam aplicados corretamente.
5. Teste a página de vôlei para garantir que o conteúdo e os estilos estejam funcionando corretamente. Abra a página `voley.html` em um navegador e verifique se as informações sobre o vôlei estão sendo exibidas corretamente e se os estilos personalizados estão aplicados.
